# API Node.js - Curso Balta.io

> API de loja criada a partir do curso de Node.js do Balta.io

## Tecnologias utilizadas: ##

```
- Node.js
- MongoDB
```

## Pacotes utilizados: ##

```
- debug
- express
- http
- nodemon
- mongoose
- dotenv
- guid
- sendgrid
- jsonwebtoken
```

## Run App

```
# Rodar o projeto em ambiente de Dev
npm run dev
```
